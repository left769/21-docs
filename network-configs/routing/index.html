    Налштування статичних маршрутів

    Тут зібрані команди для налаштування статичних маршрутів в Cisco IOS, Mikrotik ROS та ОС Linux за допомогою утиліти ІР

    1. Налаштування статичних маршрутів

    Cisco IOS: Налаштування статичного маршруту

    ip route 172.16.0.0 255.240.0.0 10.0.0.2

    Опціонально, може додаватись метрика (число від 1 до 255) для визначення пріоритету, чим менша метрика, тим більш пріоритетним ж маршрут

    ip route 0.0.0.0 0.0.0.0 10.0.0.2 7

    Налаштування шлюза останньої надії

    ip default-gateway 10.0.0.1

    Mikrotik ROS:

    /ip/route/add dst-address=0.0.0.0/0 gateway=10.0.0.2

    Linux

    sudo ip route add default via 10.0.0.2 dev eth0

    Або

    sudo ip route add 0.0.0.0/0 via 10.0.0.2 dev eth0

    2. Перегляд таблиці маршрутизації

    Cisco IOS:

    show ip route

    Mikrotik ROS

    /ip/route/print

    Linux

    ip route
